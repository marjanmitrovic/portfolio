## Objava na GitHub Pages

1. Ubaci `index.html` u root repozitorijuma.
2. Dodaj folder `images/` i fajl `og-cover.jpg`.
3. Dodaj `robots.txt` i `sitemap.xml`.
4. Push na GitHub.
5. U GitHub repozitorijumu idi na:
   - **Settings**
   - **Pages**
   - kao source izaberi **Deploy from a branch**
   - branch: **main**
   - folder: **/root**
